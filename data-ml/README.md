# data-ml

This folder contains all data, notebooks, and scripts related to data preparation,
feature engineering, and machine learning for NutriBudget.

- `raw/` – raw datasets downloaded from external sources (not committed, usually large).
- `01_explore_grocerydb.ipynb` – initial exploration of GroceryDB CleanedData.
- `02_prepare_features.ipynb` – feature engineering (scores, cleaned table).
- `03_clusters.ipynb` – clustering into health personas.
- `NOTES.md` – documentation of columns, formulas, and decisions.
